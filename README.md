## About Me

Software engineer based in **Nairobi, Kenya**, with a **BSc in Computer Technology (Second Class Upper Division)** from **Jomo Kenyatta University of Agriculture and Technology (JKUAT)**. I build and ship production-grade full-stack web and mobile applications across logistics, e-commerce, geospatial systems, and academic domains.

My work spans end-to-end ownership — database design, API architecture, payment integration, containerized deployment, and team leadership. I have hands-on experience with Stripe, PayPal, and M-Pesa Daraja 2.0, CI/CD via GitHub Actions, and geospatial engineering with GeoDjango and OSRM.
